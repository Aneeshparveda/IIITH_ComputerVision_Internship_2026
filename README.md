# Online Summer Internship 2026

## Overview
This repository contains my work from the Online Summer Internship 2026. The project focuses on building a basic computer vision pipeline including video processing, object detection, segmentation, dataset preparation, and model training.

## Week 1 – Video Processing
- Extracted frames from a video using FFmpeg  
- Reconstructed video from frames  
- Added audio to the video  

## Week 2 – Object Detection
- Created Python virtual environment using venv  
- Used Ultralytics YOLO for object detection  
- Applied detection on images and converted them into video  

## Week 3 – Segmentation & Video Comparison
- Performed semantic segmentation on images  
- Generated segmented video  
- Combined original, detection, and segmentation videos into one  

## Week 4 – Dataset Preparation
- Explored YOLO dataset structure and label format  
- Created dataset using Label Studio  
- Generated train and validation splits  

## Week 5 – Sickle Cell Detection
- Annotated dataset and prepared YOLO labels  
- Trained YOLO model on custom dataset  
- Tested model on new images  
- Generated prediction videos and comparison outputs  

## Tools Used
- Python  
- FFmpeg  
- Ultralytics YOLO  
- OpenCV  
- Label Studio  

## Summary
This internship helped me understand the complete workflow of a computer vision project, from data processing to model training and result generation.
